# ruijie-zhu.github.io

Personal academic homepage of Rui-Jie Zhu, served as plain static HTML on GitHub Pages (`.nojekyll`, no build step).

- `index.html` — homepage
- `publications/index.html` — publication list (maintained by hand, by year)
- `css/blog.css` — the only stylesheet
- `images/` — profile photo and logos

Design adapted from [Tianyu Gao](https://gaotianyu.xyz)'s website, in turn adapted from [Gregory Gunderson](http://gregorygunderson.com/).
